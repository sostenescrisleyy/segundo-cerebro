# Font Pairing — Combinando Tipografias

## Princípios Fundamentais

### 1. Contraste, não conflito
Pares que funcionam criam *tensão visual interessante*. Pares que não funcionam criam *ruído visual*.

**Regra de ouro:** Use no máximo 2–3 famílias tipográficas por projeto. Em identidades visuais, geralmente 1 principal + 1 auxiliar é suficiente.

### 2. Categorias complementares
As combinações mais seguras misturam categorias diferentes:

| Função | Família A | Família B |
|---|---|---|
| Título | Serifa | Sem Serifa |
| Corpo | Sem Serifa | Serifa |

### 3. Hierarquia clara
Um estilo domina (display/título), o outro suporta (corpo/legenda). Nunca coloque dois estilos decorativos competindo na mesma peça.

---

## Combinações Clássicas e Funcionais

### Serifa Moderna + Sem Serifa Geométrica
Sofisticação + modernidade. Muito usada em moda e beleza.
- Didot + Futura
- Bodoni Moda + Poppins
- Playfair Display + Montserrat

### Serifa Humanista + Sem Serifa Humanista
Orgânico + legível. Funciona bem em editoriais e marcas acolhedoras.
- Garamond + Gill Sans
- Cormorant Garamond + Cabin
- EB Garamond + Myriad

### Serifa Transicional + Sem Serifa Neo-Grotesca
Clássico + neutro. O par mais versátil, funciona em quase tudo.
- Baskerville + Helvetica
- Libre Baskerville + Inter
- Georgia + Inter

### Slab + Sem Serifa Bold
Robustez + impacto. Ideal para marcas fortes, industriais ou esportivas.
- Bitter + Montserrat
- Zilla Slab + Oswald
- Roboto Slab + Roboto

### Script (destaque) + Sem Serifa (corpo)
Elegância + legibilidade. Script nunca como fonte de corpo — apenas como elemento de destaque.
- Great Vibes + Montserrat
- Dancing Script + Lato
- Pacifico + Nunito

---

## Combinações a Evitar

❌ **Dois estilos decorativos juntos** (Script + Blackletter, Hand Lettering + Script)  
❌ **Duas serifas da mesma subcategoria** (Times New Roman + Baskerville — parecidos demais)  
❌ **Script como fonte de corpo de texto** — ilegível em tamanhos pequenos  
❌ **Blackletter em corpo de texto** — muito difícil de ler  
❌ **Mais de 3 fontes na mesma peça** sem hierarquia clara  

---

## Processo para Escolher o Par

1. **Defina a fonte principal** (geralmente para títulos/display) — ela carrega a personalidade
2. **Identifique o que falta** — se a principal é muito decorativa, o par precisa ser neutro e funcional
3. **Teste em contexto real** — veja como ficam juntas em tamanho real, não só no comparador de fontes
4. **Verifique pesos disponíveis** — a família precisa ter Regular, Bold e Italic no mínimo
5. **Teste em tamanhos extremos** — a fonte de corpo precisa funcionar em 10–14px

---

## Fontes que Funcionam Bem Sozinhas (Super Famílias)

Algumas famílias são tão completas que uma única família basta para o projeto inteiro, usando pesos e estilos diferentes:

- **Freight** (Display + Text + Micro) — sofisticada
- **FF Meta** (Serif + Sans) — corporativa/editorial
- **Playfair Display + Playfair Display SC** — elegante
- **Merriweather + Merriweather Sans** — editorial/editorial digital
- **Source Serif + Source Sans + Source Code** — tech/editorial
